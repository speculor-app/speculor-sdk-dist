# Security Policy

## Reporting a vulnerability

If you find a security issue in the Speculor SDK, the engine, or anything you
download from this repository's Releases, please report it privately — do **not**
open a public issue.

Email **security@speculor.app** (or **contact@speculor.app**) with:

- a description of the issue and its impact,
- steps to reproduce (a minimal plugin or `.speculor` graph if relevant),
- the SDK / engine version (`SpeculorSDK-<version>-…`) and your platform.

We'll acknowledge your report, keep you updated on the fix, and credit you in the
release notes if you'd like.

## Scope

This repo is the public distribution point (docs + release bundles). The SDK and
engine source live in private repositories; report issues here and we'll route
them internally.
